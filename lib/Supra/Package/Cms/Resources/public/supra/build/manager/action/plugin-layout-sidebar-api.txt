API for Sidebar:

Attributes:
	headerNode
		Header node
		
	contentNode
		Content node
		
	headerVisible
		Header visibility
		
	controlButton
		Header control button, "Done"
		
	frozen
		If frozen then toolbar buttons will not be hidden when sidebar
		visibility changes; execute function will not be called
		when action is executed; hide function will not be called when
		sidebar is hidden
		
	backButton
		"Back" button widget
		
	titleNode
		Sidebar title node
		
	title
		Sidebar title
		
	iconNode
		Sidebar icon node
		
	icon
		Sidebar icon
