Manager.Page
	Methods:
		getPageIdFromPath - loads page ID from path and returns page id to callback
		getPageData - returns page data
		isPage - returns true if currently edited page is not template
		isTemplate - returns true if currently edited page is template
		getType - returns 'page' is currently editing page or 'template' if editing template
		
		loadPage - load page data
		publishPage - publish current page
		unlockPage - unlock page
		lockPage - lock page
		deleteCurrentPage - delete current page
		deletePage - delete page
		createPage - create new page and returns page data to callback
		updatePage - Update page data and returns new page data to callback

Manager.PageContent
	Events:
		...
		
	Properties:
		iframe - iframe wrapper object, IframeHandler
		contents - content handling object, ContentHandler

		getContent - returns ContentHandler object


IframeHandler
	Events:
		reset - iframe content is reloading
		load  - all content has finished loading
		ready - all content loaded and blocks initialized
	
	Attributes:
		doc - document
		win - window
		body - body (getter only)
		overlayVisible - shim for iframe
	
	Methods:
		addScript - add js file to the iframe content
		addStyleSheet - add css file to the iframe content
		
		setHTML - set page source (HTML)
		
		createContent - creates ContentHandler object instance
		destroyContent - destroys ContentHandler object instance


ContentHandler
	Events:
		resize 
	
	Attributes:
		iframe - IframeHandler instance
		doc - document
		win - window
		body - document body
		
		disabled - disabled state
		
		activeChild
	
	Methods:
		getChildById - returns block by ID
		getAllChildren - returns all children and children of children
		
		removeChild - removes block and sends update to server
		
		getNewChildData - requests new child data from server
		


ContentProto
	Attributes:
		win - window
		doc - document
		body - document body
		
		data
		
		editable - is content editable?
		editing - current editing state
		changed - has content changed
		
		draggable - content can be dragged and droped (order)
		highlight - current highlight state
		highlightOverlay - current overlay highlight state
		
		parent
		super
	
	Properties:
		children - Array of children
		children_order - Array of children IDs in correct order
	
	Methods:
		saveChildrenOrder - save child order and send update to server
		saveProperties - save properties and send update to server
		
		isChildTypeAllowed - checks if given child type is allowed
		addChild - adds block and sends update to server
		removeChild - removes block and sends update to server
		getChildById - returns child by ID
		getAllChildren - returns all children
		
		getId - returns content ID
		getNode - returns container node
		getNodeId - returns container node ID
		getBlockType - return block type
		getBlockTitle - returns block title
		getBlockInfo - returns block information
		
		isClosed - closed content children can't be added, removed or moved
		isParentClosed
		
		processData - remove all uneeded before it will be sent to server
