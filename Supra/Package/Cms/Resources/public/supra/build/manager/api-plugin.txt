Action.PluginBase API:
PROPERTIES:
	host
		Action, which contains this plugin
	
	instances
		List of widget instances
	
	placeholders
		Placeholder nodes (host container nodes)

METHODS:
	addWidget(widget, prefix)
		Add widget to plugin and host
		widget - Widget instance
		prefix - Optional. Widget ID
		
	getWidget(id)
		Returns widget by ID
	
	getWidgets()
		Returns all widget instances created/generated by plugin

CONSTRUCTOR:
	initilalize()
		Initialize plugin. This function should be overwritten
	
	execute()
		Execute plugin. This function should be overwritten.
